# 저장소 생성과 클론

## 목표
- 새 저장소를 만들거나 기존 저장소를 클론하는 방법 정리
- 기본 설정(사용자 정보, 기본 브랜치 등)을 기록

## 핵심 개념
- `git init` vs `git clone`
- 로컬/원격 저장소의 관계
- 초기 설정(사용자 이름, 이메일)

## 주요 명령어
- `git init`
- `git clone`
- `git config`

## 예시
```sh
git init
git clone https://github.com/USER/REPO.git
git config --global user.name "YOUR_NAME"
git config --global user.email "YOUR_EMAIL"
```
