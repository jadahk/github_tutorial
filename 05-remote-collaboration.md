# 원격 저장소와 협업

## 목표
- 원격 저장소 추가, 동기화 흐름 정리
- 협업 시 자주 쓰는 명령어 정리

## 핵심 개념
- fetch vs pull
- push 권한과 브랜치 정책
- upstream/tracking 브랜치

## 주요 명령어
- `git remote`
- `git fetch`
- `git pull`
- `git push`

## 예시
```sh
git remote -v
git fetch origin
git pull origin main
git push origin feature/login
```
